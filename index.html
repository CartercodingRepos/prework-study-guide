Prework Study Guide

This Study Guide is for HTML, CSS, Git and JavaScript.

HTML

  • The head element contains the title, a link to style.css and meta data for the webpage.
  • The header element contains the h1 tag.
  • The body element represents the visible content shown to the user broken down into articles, sections, asides and a footer.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  1. Git Bash: to open commandline.
  2. Git status: checks what branch we are currently on
  3. Git checkout -b branch-name: creates a new branch and switches to it
  4. Git add -A: adds the starting code to the branch from main.
  5. Now we code!
  6. Git commit -m "COMMENT ON YOUR CODING CHANGES BETWEEN THE QUOTE MARKS HERE"
  7. Git pull origin main: check local branch nad main branch are in sync.
  8. Git push origin feature/NAME-OF-FEATURE-BRANCH-HERE
  9. Go to github, select the branch feature/NAME-OF-FEATURE-BRANCH-HERE, select "compare & pull request", give pull request a descriptive title like "added starter code for project", "create pull request", make first pull request, merge by clicking "merge pull request" to merge feature branch into main branch. Finally, close any Issues in github repo.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An 'if' conditional is a decision that meets a condition or is true.
  • An 'if else' is used when there is more than one condition.
  • An array is a single variable holding a group of data typically related in someway. This can also be different types of data such as strings, numbers, etc. Remember, strings are always in quotation marks.
  • Data in an array has an index number. The first item in any array has an index of 0.
  • Use 'for' as a loop. Loops have three parts. Loops have a starting point variable of x with a value of 0, e.g. x = 0; The second statement is a condition. The conditional determines the length and uses an array. For example an array of shapes would be: x < shapes.length; . The final statement increases the array by 1 each time e.g. x++ increases by 1 the same as x+1. Therefore, together e.g. for (var x = 0; x < shapes.length; x++) {console.log (shapes[x]);}.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
